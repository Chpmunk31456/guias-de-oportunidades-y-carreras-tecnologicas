# Cómo elegir una carrera tecnológica

Esta guía breve ayuda a comparar opciones sin asumir que existe una carrera perfecta para todas las personas. No promete empleo ni reemplaza orientación profesional, médica, legal o educativa.

## 1. Empiece por sus condiciones reales

Anote respuestas sencillas:

- ¿Cuántas horas por semana puede estudiar de forma sostenible?
- ¿Tiene computadora e Internet confiables?
- ¿Necesita trabajar mientras aprende?
- ¿Prefiere analizar, construir, investigar, documentar, enseñar o coordinar?
- ¿Qué adaptaciones o apoyos le permiten trabajar mejor?
- ¿Busca trabajo remoto, presencial o cualquiera de los dos?
- ¿Qué idiomas utiliza hoy y cuáles desea desarrollar?

No convierta una barrera actual en una conclusión permanente. Identifique también apoyos, alternativas y rutas de entrada.

## 2. Compare dos o tres opciones

Para cada carrera, califique de 1 a 5:

| Criterio | Pregunta |
|---|---|
| Interés | ¿Quiero aprender y practicar estas tareas? |
| Acceso | ¿Puedo comenzar con recursos y equipos que ya tengo? |
| Habilidades transferibles | ¿Puedo aprovechar experiencia previa? |
| Adaptaciones | ¿Puedo realizar el trabajo con apoyos razonables? |
| Evidencia | ¿Puedo construir un proyecto seguro y demostrable? |
| Mercado local | ¿Existen vacantes reales en mi región o modalidad? |
| Tiempo | ¿La ruta inicial cabe en mis responsabilidades actuales? |

La puntuación ayuda a conversar; no decide por usted.

## 3. Haga un experimento gratuito de 14 días

1. Lea una guía individual.
2. Complete una introducción gratuita de una fuente oficial o reconocida.
3. Practique una tarea pequeña con datos públicos, ficticios o autorizados.
4. Documente qué disfrutó, qué fue difícil y qué apoyo necesitó.
5. Revise cinco ofertas reales para distinguir requisitos comunes de listas poco realistas.

No compre un programa costoso antes de probar la actividad y verificar sus condiciones, reembolsos, resultados y reputación.

## 4. Tome una decisión de 30 días

Elija una opción para explorar durante 30 días y defina:

- una habilidad fundamental;
- una práctica pequeña por semana;
- un proyecto de portafolio seguro;
- una fecha de revisión;
- una señal para continuar, ajustar o cambiar de dirección.

Cambiar de dirección con nueva evidencia no es fracasar.

## Señales de alerta

- Garantía de empleo o salario.
- Presión para pagar inmediatamente.
- Costos, deuda o condiciones poco claros.
- Solicitud de trabajar sin autorización en sistemas ajenos.
- Uso de datos personales o confidenciales en herramientas no aprobadas.
- Recomendación de inventar experiencia, títulos o certificaciones.
