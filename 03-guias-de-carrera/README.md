# Guías de Carreras Tecnológicas

Recursos gratuitos y prácticos en español para ayudar a las personas a explorar, preparar y comenzar una carrera en tecnología.

Esta sección forma parte del proyecto **Guías de Oportunidades y Carreras Tecnológicas**.

## Comienza aquí

Próximamente esta sección incluirá:

- Las 50 carreras tecnológicas del futuro
- Una guía para elegir una carrera tecnológica
- Guías individuales de carrera
- Recursos educativos gratuitos
- Herramientas de código abierto
- Proyectos de portafolio
- Orientación para GitHub
- Preparación de currículum y entrevistas
- Recursos gratuitos de inglés para tecnología
- Accesibilidad y adaptaciones
- Seguridad humana
- El principio de no causar daño
- Planes de aprendizaje de 30, 60 y 90 días

## Áreas de carrera

Las guías cubrirán carreras relacionadas con:

- Inteligencia artificial
- Ciberseguridad
- Privacidad
- Gobierno, riesgo y cumplimiento
- Datos
- Computación en la nube
- Desarrollo de software
- Robótica
- Internet de las cosas
- Infraestructura tecnológica
- Tecnología cuántica
- Energía limpia
- Accesibilidad
- Tecnología de asistencia
- Tecnología centrada en la persona

## Cómo utilizar estas guías

1. Revise la guía para elegir una carrera tecnológica.
2. Seleccione dos o tres carreras para explorar.
3. Lea las guías individuales.
4. Complete un pequeño proyecto o experimento de aprendizaje.
5. Utilice primero los recursos gratuitos.
6. Revise su progreso después de 30 días.
7. Cambie de dirección cuando la evidencia muestre una opción más adecuada.

Cambiar de dirección no significa fracasar.

## Principio del proyecto

**No causaremos daño. Ningún resultado profesional es más importante que la seguridad, la dignidad y la vida de una persona.**

**Ayuda a una persona. Comparte una oportunidad verificada. Enseña una habilidad útil. Deja el camino más claro de como lo encontraste.**
