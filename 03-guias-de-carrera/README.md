## Qué incluye cada guía

Las guías pueden incluir:

- Descripción de la carrera
- Responsabilidades comunes
- Puestos iniciales y avanzados
- Habilidades técnicas
- Habilidades humanas y de comunicación
- Rutas con y sin título universitario
- Recursos educativos gratuitos
- Herramientas de código abierto
- Proyectos de portafolio
- Evidencia para GitHub
- Certificaciones opcionales
- Preparación para entrevistas
- Accesibilidad y adaptaciones
- Consideraciones éticas y legales
- Planes de aprendizaje de 30, 60 y 90 días

## Cómo utilizar estas guías

1. Lea la [guía para elegir una carrera tecnológica](elige-tu-carrera-tecnologica.md).
2. Revise [las 50 carreras tecnológicas del futuro](las-50-carreras-tecnologicas-del-futuro.md).
3. Seleccione dos o tres carreras para explorar.
4. Abra la categoría correspondiente.
5. Lea las guías individuales.
6. Complete un pequeño proyecto o experimento de aprendizaje.
7. Utilice primero los recursos gratuitos.
8. Revise su progreso después de 30 días.
9. Cambie de dirección cuando la evidencia muestre una opción más adecuada.

Cambiar de dirección no significa fracasar.

## Principio del proyecto

**No causaremos daño. Ningún resultado profesional es más importante que la seguridad, la dignidad y la vida de una persona.**

**Ayuda a una persona. Comparte una oportunidad verificada. Enseña una habilidad útil. Deja el camino más claro de cómo lo encontraste.**
