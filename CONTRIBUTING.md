# Cómo contribuir

Se agradecen correcciones factuales, enlaces oficiales actualizados, mejoras de accesibilidad y propuestas que mantengan el enfoque gratuito, ético y centrado en la persona.

## Antes de enviar una propuesta

- Verifique la información en una fuente oficial.
- Indique la fecha de verificación.
- Explique qué cambia y por qué.
- No copie material protegido sin permiso.
- No incluya datos personales, credenciales ni información confidencial.
- No añada promesas de empleo, salario, admisión o certificación.
- Distinga claramente entre recursos gratuitos, auditoría gratuita y opciones de pago.

## Corrección sugerida

Abra un issue con:

- **Archivo o guía:**
- **Sección:**
- **Problema:**
- **Corrección propuesta:**
- **Fuente oficial:**
- **Fecha de verificación:**
- **Impacto de accesibilidad o seguridad, si existe:**

## Cambios mediante pull request

1. Mantenga cada cambio limitado a un propósito claro.
2. Conserve la estructura y la licencia del proyecto.
3. Revise ortografía, enlaces, encabezados y navegación.
4. Explique las verificaciones realizadas.
5. Declare el uso material de IA y revise personalmente el resultado.

La aceptación de una contribución no implica respaldo de un empleador, curso, proveedor o certificación.
